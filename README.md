# Alexa for MacOS/OSX
Control Alexa, via [alexa.amazon.com](http://alexa.amazon.com), in a "native" feeling app.

![Screenshot](screenshot.png)

# Installation
[Download & Move Alexa.app](Alexa.app) to your Applications folder then run.
